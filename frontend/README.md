
# 🎨 LeetLens Frontend

Frontend application for  **LeetLens** , a modern interview intelligence platform for exploring company-wise LeetCode questions and interview trends.

Built with React, Vite, and TailwindCSS.

---

## 🚀 Tech Stack

* React
* Vite
* TailwindCSS
* Axios

---

## 📦 Installation

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:5173
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🌐 Backend API

The frontend communicates with the FastAPI backend.

Default backend URL:

```text
http://127.0.0.1:8000
```

Make sure the backend server is running before starting the frontend.

---

## ✨ Features

* Real-time LeetCode question search
* Search by problem number
* Search by title
* Company-wise interview insights
* Difficulty filtering
* Responsive UI
* Dark mode design
